# 골목고양이 생존기 — Game Design Document

**버전**: 0.1 (MVP 기준)  
**장르**: 2D 도트 횡스크롤 생존 로그라이트  
**플랫폼**: Android 우선, 추후 iOS  
**엔진**: Unity 2D (URP)  
**레퍼런스 느낌**: 별의 커비 (귀엽고 직관적)

---

## 1. 핵심 콘셉트

사람이었던 주인공이 의문의 사건으로 고양이가 되어, **사람의 기억을 가진 채 골목에서 살아남는** 게임.  
플레이어는 횡스크롤 골목 맵에서 좌우로 이동하며 먹이·돈·경험치를 모으고, 랜덤 사건에 대응하며 경험치 100%를 달성해 다음 스테이지로 나아간다.

---

## 2. 세계관 & 프롤로그

```
사람이 골목을 걷는다
→ 의문의 사건 발생 (연출만, 원인은 미공개)
→ 정신을 잃음
→ 낡은 박스 안에서 고양이로 깨어남
→ 앞발을 보고 상황 인식
→ 배고픔 UI 등장
→ 첫 생존 시작
```

주인공은 **사람의 기억이 남아 있어** 돈의 가치, 가게 구조, 사람의 행동 패턴을 이해한다.  
장기 목표 중 하나는 "왜 고양이가 되었는가"의 단서를 모으는 것이지만, MVP에서는 연출 암시 수준으로만 처리한다.

---

## 3. 게임 목표

| 구분 | 내용 |
|------|------|
| **승리 조건** | 경험치 게이지 100% 달성 → 다음 스테이지 |
| **패배 조건** | 체력 0 또는 배고픔 0 후 일정 시간 경과 |

### 단기
- 횡스크롤 맵에서 좌우 이동하며 생존
- 먹이·돈 수집 / 사건 회피·해결 / 경험치 100% 달성

### 장기
- 높은 점수 달성
- 여러 맵 해금
- 고양이 능력 강화
- "왜 고양이가 되었는가" 단서 수집

---

## 4. 기본 플레이 루프

```
스테이지 시작
→ 횡스크롤 맵 좌우 이동
→ 먹이 / 돈 / 경험치 수집
→ 랜덤 사건 발생 (사람 / 개 / 고양이 / 장애물)
→ 도망 / 숨기 / 훔치기 / 싸우기 선택
→ 피버 게이지 충전
→ (선택) 피버타임 발동
→ 경험치 100% 달성
→ 클리어 연출 (Win Cheer)
→ 점수 정산 & 자동 저장
→ 다음 스테이지
```

## 5. 조작 (모바일)

| 입력 | 기능 |
|------|------|
| 가상 조이스틱 or 좌/우 버튼 | 좌우 이동 |
| 버튼 A | 대시 / 회피 (Spining) |
| 버튼 B | 먹기 / 줍기 / 상호작용 |
| 버튼 C | 숨기 / 은신 (Ducking) |
| 버튼 D | 피버타임 발동 |

PC 테스트용: 방향키 이동, Z=대시, X=상호작용, C=숨기, V=피버

## 6. 스테이지 구성

### 횡스크롤 맵 레이아웃 (공통)
```
[시작 박스] → [골목길] → [쓰레기통] → [사람 구역] → [먹이 구역] → [대치 구역] → [클리어 지점]
```
- 카메라는 플레이어를 좌우 추적
- 구역마다 사건 트리거 배치
- 사건 해결/회피 시 경험치 획득

### 스테이지별 시나리오

#### Stage 1 — 시골 골목 (튜토리얼)
- **분위기**: 낮, 여유로운 시골 골목
- **오브젝트**: 낮은 담벼락, 나무 상자, 풀숲, 쓰레기봉투
- **빌런**: 느린 동네 사람(感知 범위 좁음), 약한 고양이
- **학습**: 기본 이동·점프·먹이 획득·사람 피하기
- **클리어 핵심**: 생선 획득 + 사람 회피

#### Stage 2 — 작은 마을
- **분위기**: 낮, 골목 주택가
- **오브젝트**: 전봇대, 쓰레기통, 자전거, 낮은 담벼락
- **빌런**: 왕복 골목 주민, 쓰레기통 지키는 고양이
- **학습**: 숨기(Ducking) + 쓰레기통 상호작용
- **클리어 핵심**: 탐색 + 감지 범위 피하기

#### Stage 3 — 시장 골목
- **분위기**: 낮, 활기찬 재래시장
- **오브젝트**: 생선 상자, 과일 박스, 가게 좌판, 손님 NPC
- **빌런**: 생선가게 주인(빠른 추격), 빠른 경쟁 고양이
- **학습**: 타이밍 훔치기 + 빠른 도망
- **클리어 핵심**: 생선 훔치기 + 경쟁 고양이보다 먹이 확보

#### Stage 4 — 아파트 단지
- **분위기**: 낮, 복잡한 아파트 단지
- **오브젝트**: 주차장, 화단, 계단, 경비실, 자전거 도로
- **빌런**: 넓은 감지 범위 경비원, 불규칙 아이들, 자전거 장애물
- **학습**: 은신 + 타이밍 이동
- **클리어 핵심**: 은신 + 자전거 회피

#### Stage 5 — 번화가 골목
- **분위기**: 저녁, 번화한 도심
- **오브젝트**: 편의점, 배달 오토바이, 도로, 좁은 골목
- **빌런**: 개(빠른 추격), 배달 오토바이(충돌 위험), 무리 고양이
- **학습**: 복합 위험 대응 + 피버타임 활용
- **클리어 핵심**: 개 회피 + 도로 통과 + 피버타임

#### Stage 6 — 밤거리 (최종)
- **분위기**: 밤, 시야 제한
- **오브젝트**: 깜빡이는 가로등, 좁은 통로, 버려진 건물, 높은 담벼락
- **빌런**: 보스 고양이(넓은 공격 판정)
- **특수 요소**: 화면 일부 어둠 처리 → 위험 요소 늦게 발견
- **클리어 핵심**: 피버타임으로 보스 구역 돌파

## 10. 피버타임

피버 게이지가 가득 차면 발동. "사람의 판단력 + 고양이 본능이 폭발하는 상태."

| 효과 |
|------|
| 이동속도 증가 |
| 점수 2배 |
| 돈 획득량 증가 |
| 사람 감지 범위 감소 |
| 사람 물건 훔치기 가능 |
| 다른 고양이 먹이 훔치기 가능 |
| 짧은 무적 시간 |

## 11. 적 & 위험 요소

### 사람 NPC
| 종류 | 행동 패턴 |
|------|-----------|
| 일반 사람 | 가까이 가면 쫓아냄 (느림) |
| 아이 | 불규칙하게 뛰어다님 |
| 가게 주인 | 먹이 근처 감시, 발견 시 빠른 추격 |
| 경비원 | 넓은 감지 범위, 오래 추격 |
| 착한 사람 | 먹이를 줌 |

### 동물 & 장애물
| 종류 | 행동 패턴 |
|------|-----------|
| 약한 고양이 | 먹이 경쟁, 가까이 가면 위협 |
| 보스 고양이 | 구역 지킴, 넓은 공격 판정 |
| 개 | 발견 시 빠른 추격 |
| 자동차·오토바이 | 도로 구간 충돌 위험 |
| 자전거 | 빠르게 지나가는 장애물 |

## 12. 사건 시스템

| 사건 | 구현 방식 |
|------|-----------|
| 사람 등장 | 감지 범위 내 진입 시 추격 → Ducking / 대시로 회피 |
| 쓰레기통 발견 | 상호작용 버튼 → 랜덤 아이템 |
| 생선 상자 발견 | 상호작용 → 먹이 + 경험치 (가게 주인 연동) |
| 다른 고양이 대치 | Attack/Hit 간단 전투 |
| 개 등장 | 추격형 장애물 → 숨기 or 높은 곳 점프 |
| 높은 장애물 | Jump or Ladder |
| 좁은 통로 | Ducking Move 통과 |
| 휴식 장소 | Asleep → 체력/배고픔 일부 회복 |
| 착한 사람 | 접근 → 먹이 획득 |
| 자동차 도로 | 타이밍 이동으로 통과 |

## 17. 아트 방향

- **스타일**: 귀여운 2D 도트 (32×32 기준)
- **배경**: 횡스크롤 골목 (전봇대, 쓰레기통, 담벼락, 시장 좌판, 박스, 자동차)
- **진행감**: 맵이 진행될수록 복잡하고 위험한 분위기

### 보유 스프라이트 → 사용처

|-----------|--------|
| Idle | 기본 대기 |

| Ducking / Ducking Idle / Ducking Move | 숨기·낮은 자세 이동 |
| Attack | 고양이 대치 공격 |
| Hit | 피해 받음 |
| Dead | 게임오버 |
| Asleep | 휴식·회복 |
| Ladder | 전봇대·담벼락 오르기 |
| Against Wall | 벽에 붙기·은신 |

## 19. 개발 우선순위

### 1차 MVP
```
프롤로그 연출
+ 시골 골목 횡스크롤 1맵
+ 모바일 좌우 이동
+ 걷기 / 달리기 / 점프 / 착지
+ 먹이·돈 획득
+ 배고픔 감소
+ 사람 등장·추격 사건
+ 숨기 / 대시로 회피
+ 경험치 100% 달성
+ 스테이지 클리어 연출
+ 자동 저장
```

### 2차
```
다른 고양이 대치 (Attack / Hit)
+ 쓰레기통 상호작용
+ 생선 상자 훔치기
+ 피버타임
+ 휴식 장소
```

### 3차
```
사다리 / 전봇대 오르기
+ 좁은 통로 통과
+ 개 추격 사건
+ 스테이지 2~3
+ 업그레이드 시스템
+ JSON 저장
```

### 4차
```
스테이지 4~6
+ 보스 고양이
+ 성장 시스템 완성
+ 스킨 시스템
+ 메인 메뉴 완성
```
