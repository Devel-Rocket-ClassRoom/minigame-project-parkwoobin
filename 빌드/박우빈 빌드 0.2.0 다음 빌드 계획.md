## 1. 게임 오버 창

### 현재 상태
HP 0 → 사망 애니메이션 재생 후 아무것도 없음. GameOver 상태 전환만 됨.

### 목표
사망 직후 게임 오버 UI가 나타나고, 재시작 또는 타이틀로 돌아갈 수 있는 흐름 완성.

### 연출 흐름

```
플레이어 HP 0
  → PlayerController.Die() 호출
  → 사망 애니메이션 재생 (기존)
  → GameManager.GameOver() → GameState.GameOver
  → 1.5초 딜레이
  → 게임 오버 UI Fade In
     - "GAME OVER" 텍스트
     - [다시 시작] 버튼 → 현재 씬 재로드
     - [타이틀로] 버튼 → Main 씬 전환
```

### 작업 항목

- [ ] UI Canvas에 게임 오버 패널 추가 (전체화면 반투명 검정 배경)
- [ ] "GAME OVER" 텍스트 + [다시 시작] / [타이틀로] 버튼 배치
- [ ] `GameManager.OnStateChanged → GameOver` 구독 → 딜레이 후 패널 Fade In
- [ ] [다시 시작] → `SceneManager.LoadScene(현재 씬)` 후 상태 초기화
- [ ] [타이틀로] → `SceneManager.LoadScene("Main")`
- [ ] 게임 오버 중 플레이어 입력 차단 (이미 `_isDead`로 차단됨 — 확인만)
- [ ] Fade In 연출: `CanvasGroup.alpha` 코루틴 (0 → 1, 0.5초)

### 주의
- `GameManager`는 DontDestroyOnLoad → 씬 재로드 후 상태를 `Playing`으로 초기화해야 함
- 사망 애니메이션 길이만큼 딜레이를 맞출 것 (현재 클립 길이 확인 필요)

## 2. 게임 세이브

### 현재 상태
`SaveManager.cs` 파일은 존재하나 뼈대만 있고 실제 저장 로직 없음.

### 목표
게임 오버 또는 스테이지 클리어 시 진행 상황 자동 저장.

### 저장 데이터 구조

```csharp
[System.Serializable]
public class SaveData
{
    public int   unlockedStage;   // 해금된 최대 스테이지 번호 (1~4)
    public int   bestScore;       // 최고 점수 (ScoreManager 구현 후 연동)
    public int   totalCoin;       // 누적 코인
    public float totalPlayTime;   // 총 플레이 시간 (초)
}
```

### PlayerPrefs 키 상수

```csharp
public static class SaveKeys
{
    public const string UnlockedStage = "unlocked_stage";
    public const string BestScore     = "best_score";
    public const string TotalCoin     = "total_coin";
    public const string TotalPlayTime = "total_play_time";
}
```

### 저장 타이밍

| 타이밍 | 트리거 |
|--------|--------|
| 스테이지 클리어 | `GameManager.OnStateChanged → StageClear` |
| 게임 오버 | `GameManager.OnStateChanged → GameOver` |
| 앱 종료 | `OnApplicationQuit()` |

### 작업 항목

- [ ] `SaveManager.cs` 에 `SaveData` 구조체 및 `SaveKeys` 상수 작성
- [ ] `Save()` — `JsonUtility.ToJson` → `PlayerPrefs.SetString`
- [ ] `Load()` — `PlayerPrefs.GetString` → `JsonUtility.FromJson`
- [ ] 첫 실행 시 기본값 처리 (저장 데이터 없을 때 unlockedStage = 1)
- [ ] `GameManager.OnStateChanged` 구독 → StageClear / GameOver 시 자동 `Save()` 호출
- [ ] `OnApplicationQuit()` 에 `Save()` 호출 추가
