### 적 AI

#### 고양이 적 (CatEnemy) — 3종
| 변형 | HP | 속도 | 데미지 | 특징 |
|------|----|------|--------|------|
| Normal | 3 | 1.8 | 1 | 주황색, 표준 AI |
| Fast | 2 | 2.5 | 1 | 갈색, 25% 확률 분노 상태 |
| Strong | 6 | 1.2 | 2 | 검은색, 느리지만 체력 높음 |

- AI 상태: Idle → Patrol → Chase → Hit
- 분노 상태: 피격 시 60% 확률 발동, 5~9초 지속, 달리기 빈도 4배 증가
- 점프: 플레이어보다 1.5u 높거나 벽 감지 시 자동 점프, 착지 예측으로 Land 모션 선행 재생

#### 인간 적 (HumanEnemy) — 3종
| 변형 | HP | 속도 | 공격 방식 | 특징 |
|------|----|------|-----------|------|
| Type1 | 4 | 2.2 | 권총 | 조준선 시각화 |
| Type2 | 6 | 2.0 | 격투 3종 | 근접 전용 |
| Type3 | 3 | 2.7 | 권총 | 빠르고 약함 |

- 총 조준: 느리게 늘어났다 빠르게 발사 (조준 라인 UI 있음)
- 근접 범위: 1.1u 이내 / 총 범위: 8u 이내 (정면만)
- 추격 반경: 스폰 위치 기준 5u (이 범위를 벗어나면 귀환)
- AI 상태: Idle → Patrol → Chase → MeleeAttack / Aim → Hit

## 5. 플레이 방법 및 조작

> **현재 키보드 기준** (모바일 버추얼 패드 미구현)

| 액션 | 입력 |
|------|------|
| 이동 | A / D 또는 ← / → |
| 점프 | Space (누르는 시간에 따라 높이 변동) |
| 대시 | Shift |
| 숨기 | S 또는 ↓ (지상) |
| 사다리 오르기 | W / S 또는 ↑ / ↓ (사다리 접촉 시) |
| 공격 | J 또는 좌측 Ctrl |
| 상자 열기 | F (범위 내, 열쇠 보유 시) |

### 기본 플레이 흐름

1. 씬 시작 → `Playing` 상태 자동 전환 (현재 프롤로그 연출 없음)
2. 적을 피하거나 공격하며 이동
3. 아이템 획득으로 배고픔 및 HP 유지
4. 열쇠 수집 → 상자 오픈 → 희귀 아이템 획득
5. HP 0 또는 배고픔 0 지속 → GameOver

### 생존 팁

- 인간 적(Type1, Type3)은 **정면에서만 총을 쏨** → 뒤로 돌아서 접근
- 고양이 Fast 타입은 **분노 상태에서 돌진** 패턴 → 점프로 회피
- 초콜릿은 배고픔을 줄이므로 **획득 금지**
- 상자 안 아이템이 가장 좋은 경향 → 열쇠 우선 수집 권장

## 6. 알려진 제한 및 주의사항

### 물리 / 충돌

- 적과 아이템 간 충돌 레이어 분리 완료 (PR #20에서 수정)
- 플레이어 공격 HitBox: 같은 프레임 내 동일 적 다중 피격 방지 처리됨
- 벽 감지: `normal.x > 0.8` 기준 — 경사 타일에서 오판정 가능

### 애니메이션

- `PlayerAnimationController`의 Eat 상태 진입 시 **이동 입력 잠금** 발생 (의도적 동작)
- 착지 모션(Land)은 착지 1프레임 전 "aboutToLand" Raycast로 선행 재생 — 플랫폼 두께가 매우 얇으면 미발동 가능

### 미니맵

- URP의 Additional Camera 자동 렌더에서 제외됨 → 미니맵이 보이지 않으면 `MinimapCamera` 컴포넌트 활성화 여부 확인

### 빌드 시 주의

- Android: `InputSystem_Actions.inputactions` 에셋 포함 확인
- URP 설정 파일(`Assets/Settings/`) 빌드에 포함 필수
- Bullet 프리팹 Material 미설정 시 매젠타(핑크) fallback으로 보임 — "Universal Render Pipeline/2D/Sprite-Unlit-Default" 셰이더 확인
