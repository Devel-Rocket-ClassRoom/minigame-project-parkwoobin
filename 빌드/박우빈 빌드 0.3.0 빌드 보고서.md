# v0.3.0 빌드 보고서
맵 2개 추가(하수구·병원 분리), 하수구 전용 거미 적 신규 구현,
이동 경로 화살표 안내 추가. 계획 대비 맵 1개 초과 달성.
---
## 1. 핵심 이슈
| 이슈 | 내용 |
|---|---|
| **맵 제작 공수** | 플랫폼마다 실제로 올라갈 수 있는지 플레이 테스트하며 배치해야 해서 예상보다 시간이 크게 소요됨. 레벨 디자인 검수 프로세스 정립 필요 |
| **오디오 리소스 탐색** | 효과음·배경음 소재 탐색 및 선별에 시간 소모. 사운드 레퍼런스 목록 사전 정리 필요 |
---
## 2. 계획 대비 결과
| 항목 | 계획 | 결과 |
|---|---|---|
| 맵 추가 | Map4(하수구) 1개 | Map4(하수구) + Map5(병원) 분리 — **+1 초과 달성** |
| 스토리 흐름 | Map3→Map4(병원) 2스테이지 | Map3→Map4→Map5 3스테이지로 재편, 스토리 자연스럽게 연결 |
| 전용 적 | 거미 추가 | 구현 완료, 일부 SFX 미적용 |
| 이동 안내 | - | 화살표 경로 안내 추가 (계획 외 추가) |
