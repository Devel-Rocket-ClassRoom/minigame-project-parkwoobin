## 2. 이번 버전(v0.2) 구현된 기능

### 2-1. 플레이어 이동 개선

#### 벽 그립 시스템 (신규)

| 조건 | 동작 |
|------|------|
| 공중 + 벽 접촉 + 벽 방향키 유지 | 중력 제거, 수직속도 0 고정 → 벽에 매달림 |
| 공중 + 벽 접촉 + 반대방향키 입력 | 0.1초 유예 타이머 시작. 그 동안 벽 방향으로 밀어서 물리 접촉 유지 |
| 유예 타이머 만료 | 낙하 시작 (`_wallGripUsed = true` → 재시작 방지) |
| 벽 방향키 재입력 | 타이머·소모 초기화 → 다시 매달리기 가능 |
| 착지 | 타이머·소모 자동 초기화 |

- 반대방향키를 계속 누르고 있어도 타이머가 무한 재시작되지 않음 (`_wallGripUsed` one-shot 패턴)
- 유예 시간 동안 스프라이트는 벽 방향 유지 (반대키 눌러도 뒤집히지 않음)

#### 벽 점프 개선

- 반대방향키를 누른 상태에서 점프해도 판정 성공 (그립 타이머 소모 방식)
- 벽 점프 후 0.25초간 수평속도 잠금 (`_wallJumpTimer`)
- 벽 점프 중 벽 방향 입력 무효화 → 튕겨나가는 방향만 허용
- 착지 체크: `_isGrounded && velocity.y ≤ 0` 조합으로 상승 중 오판정 방지

#### 공중 대시 직선 운동 (신규)

- 공중 대시 중 `gravityScale = 0`, `velocity.y = 0` 고정 → 포물선 낙하 없이 완전한 직선
- 대시 종료 시: 벽에 닿아 있으면 벽 그립으로 전환, 공중이면 `velocity.y = -2f`로 자연스럽게 낙하 시작

#### 대시 후 벽 자동 고정 (`_dashWallStick`, 신규)

- 공중 대시가 벽에 닿으면서 끝나면 자동으로 벽에 고정
- 고정 중 `gravityScale = 0`, `velocity = zero` → 벽에 붙어 정지
- 벽 점프 / 착지 / 벽 이탈 시 자동 해제

#### 계단 1타일 벽 판정 제거 (Raycast 기반)

```
플레이어 위치 → 아래 Raycast → 지면 Y 계산
지면 Y + 1.1f 높이 → 벽 방향 Raycast(0.5f)
벽 없으면 → 계단으로 판단 → _isOnWall = false
```

- 1칸짜리 턱에서 벽 매달리기가 발동되지 않음
- Physics Material(friction=0)만으로는 계단 감지 불가 → 두 방식 병행

### 2-2. 물리 재료 (PhysicsMaterial2D)

`Assets/Physics/PlayerNoFriction.physicsMaterial2D`  
Friction = 0 / Bounciness = 0 → 플레이어 Collider2D에 적용

- 점프 중 벽 방향키를 누를 때 마찰력으로 공중 부양되던 현상 해결
- 코드만으로 해결 시도 실패 이력 있음 → **빌드 포함 필수, 제거 금지**

### 2-3. Turn(회피) 모션 무적 및 총알 통과

**무적**
- Turn 발동 시 `_invincibleTimer = Mathf.Max(_invincibleTimer, turnDuration)` 설정
- 기존 피격 무적이 남아 있을 경우 더 긴 쪽을 유지
- Player ↔ Enemy 레이어 물리 충돌도 `TurnPassthroughRoutine` 코루틴으로 동시 비활성화

**총알 통과**
- `TakeDamage` 반환 타입이 `void → bool`로 변경
  - 데미지가 실제로 들어간 경우 `true` → 총알 소멸
  - 무적 상태로 데미지가 막힌 경우 `false` → **총알이 플레이어를 통과해 계속 날아감**
- Turn 회피 시 "피했다"는 연출이 시각적으로 표현됨

> ⚠️ `TakeDamage`를 호출하는 다른 스크립트 추가 시 반환값 처리 여부를 검토할 것.

### 2-4. Idle 애니메이션 무한 루프

- 이전: Idle 3회 반복 후 마지막 프레임 freeze
- 변경: freeze 로직 전체 제거 → **Idle이 멈추지 않고 계속 루프**
- 제거된 코드: `_idleFrozen`, `_lastUnfreezeFrame`, `Unfreeze()`, `AnyActionTriggerPending()`, `_actionTriggers`

## 5. 플레이 진행 방법 (현재 버전)

### 씬 실행 조건

1. Unity 에디터에서 `Assets/Scenes/TutorialMap.unity` 열기
2. 씬에 필요한 오브젝트:
   - **Player**: `PlayerController`, `PlayerAnimationController`, `PlayerInput`, `Rigidbody2D`, `Collider2D`
   - **Player Collider2D**: `PlayerNoFriction.physicsMaterial2D` 적용 확인
   - **GameManager**: `GameManager` (DontDestroyOnLoad)
   - **HungerSystem**, **CoinKeySystem**
   - **Camera**: `CameraFollow` (target = Player Transform)
   - **HUD Canvas**: `HUDController` + Slider/Text UI 연결

### 조작 (키보드 테스트용)

| 키 | 기능 |
|----|------|
| ← / → | 이동 |
| Space | 점프 (누르는 시간에 따라 높이 가변) |
| D | 대시 |
| ↓ | 숨기 (지상) |
| A | 공격 |
| S | 회피 (Turn) |
| A | 상자 열기 (범위 내, 열쇠 보유 시) |

### 조작 (모바일)

| 버튼 | 기능 |
|------|------|
| 조이스틱 | 이동 |
| Jump | 점프 |
| Dash | 대시 |
| 조이스틱 아래 | 숨기 |
| Attack | 공격 / 상자 열기 |
| Turn | 회피 |

### 벽 관련 조작 요령

| 상황 | 방법 |
|------|------|
| 벽 매달리기 | 공중에서 벽에 닿은 채로 벽 방향키 유지 |
| 벽 점프 | 매달린 상태에서 Jump (방향키 무관) |
| 회피 후 벽 점프 | 반대방향키 누른 후 0.1초 이내 Jump |
| 공중 대시 후 벽 고정 | 대시가 벽에 닿으면서 끝나면 자동 고정 |

### 현재 버전 플레이 가능 범위

- 좌우 이동, 점프, 대시, 숨기
- 벽 그립, 벽 점프, 공중 대시 직선 운동, 대시 후 벽 고정
- 사람/고양이 적 회피 및 공격
- Turn(회피) 모션으로 총알 통과 회피
- 아이템(음식/코인/열쇠) 획득 및 HUD 반영
- 박스 열기 (열쇠 소모)

### 현재 동작하지 않는 것

- 게임 오버 화면 (HP 0 → 사망 애니메이션 후 멈춤)
- 배고픔 0 패널티
- 사다리 타기
- 경험치, 점수, 저장/불러오기

### 생존 팁

- 인간 적은 **정면에서만 총을 쏨** → 뒤로 돌아서 접근
- **Turn 모션 중 무적 + 총알 통과** → 총알이 날아올 타이밍에 Turn으로 회피 가능
- 고양이 Fast 타입은 **분노 상태에서 돌진** → 점프로 회피
- 초콜릿은 배고픔 감소 아이템 → 획득 금지
- 1칸짜리 턱에서는 벽 매달리기 비활성화 (계단 감지 로직)
